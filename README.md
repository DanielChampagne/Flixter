A two-sided, video-streaming marketplace platform that features credit card payment capabilities, user role management, complex user interfaces, and advanced database relationships.

#### Built using:
- Ruby v2.5.3
- HTML5
- CSS
- Bootstrap 4.3.1
- JQuery
- Popper JS
- Font Awesome
- Stripe API

#### Ruby Gems used:
- Carrierwave
- Figaro
- Simple-Form
- Devise
- Mini-magick
